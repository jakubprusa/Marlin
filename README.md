# Marlin 3D Printer Firmware
This is my customization of Marlin firmware. Firmware currently run my Rebelix 3D printer. 

## Setup 
Printer has following setup

 - Megatronic v3.0 board
 - Jhead hotend
 - TMC 2130 drivers
 - 	Probe
 - Tiny Oled I2C display
 - Rotary Encoder
